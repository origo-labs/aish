AISH wraps commands in a PTY, logs everything, and surfaces the exact failure region that matters. You still run normal tools. AISH just makes the output usable when it counts.

PTY-True Execution

Progress bars, colors, and interactive behavior stay intact because commands run in a real PTY.

Detector-Aware Excerpts

Pytest, Jest, Vitest, Cargo, Go, Terraform, Docker, Kubectl, and more. Tool-specific patterns win over generic fallbacks.

Shell Native

Works with explicit ai wrapping plus optional command shims in Bash and zsh.

Artifacts You Can Trust

Every run gets structured metadata, digest, relevant excerpt, and full raw log. Retention keeps storage bounded.

Fast Debug Loop

aish-run --last shows context immediately. --open jumps straight into full logs with your pager.

Built for engineers who live in terminals: fewer scroll marathons, faster root-cause, same command workflow.
